### Conversor de PDF para Markdown.

---
Para esse projeto use PDFs na linguagens:
- Português
- Inglês

> Use somente PDFs digitalizados

---

Sem suporte a arquivos:

- Escaneados
- Imagens

---

para converter coloque o `arquivo.pdf` no diretório:
```bash
\input
```

Depois execute:

```bash
pyhton converter.py \input\arquivo.pdf
```

Verifique a saída do arquivo em:

```bash
\output
```

seu `arquivo.md`
